# Alpaca Trading Toolkit (Python)

A simple, transparent trading toolkit for Alpaca users.
Designed for learning, testing strategies, and understanding risk.

## Features
- Real Alpaca market data
- EMA + RSI strategy
- Risk-based position sizing
- Trade decision logging (CSV)
- Paper trading compatible

## What this is NOT
- No guaranteed profits
- No HFT
- No financial advice
- No automated live trading by default

## Requirements
- Python 3.10+
- Alpaca account (Paper)
